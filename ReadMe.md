# XiaoMI Router Onekey SSH
# 小米路由一键SSH脚本
# 资源收集整理自网上
#### (此脚本仅供 2.8.16 或之前 , 后来小米修复了漏洞 )
#### 附件附带2.8.16镜像，可以选择手动升级，镜像并不会清空硬盘数据，放心降级/升级
#### PS:小米路由IP应该为 192.168.31.1


# Usage:

### 方法1:

#### 使用Python3.0+ , Requests库 

###### R1D测试通过

```python
python mi.py
```

然后输入Stock号


### 方法2：
#### windows系统下适用EnableSSH.exe 一键
适用方法同上，输入Stock号
